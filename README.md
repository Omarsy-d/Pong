 # Pong
 A simple game of pong 

---


### Game Instructions
#### Objective:

Prevent the ball from passing your paddle.
Score points by making the ball pass the AI's paddle.

##### Controls:

##### Player 1 (You):
Use the UP and DOWN arrow keys or W and S keys to move your paddle.

##### AI:
